    Introduction

    Oncoscape is a web application that hosts an integrated suite of analysis tools for users to explore hypotheses related to molecular and clinical data in order to better understand cancer biology and treatment options. Oncoscape is as an SPA – a single page web application – using JavaScript in the browser and R (primarily) on the backend server for statistical calculations. For more detailed information, please read wiki.

    Authentication

    Oncoscape API authentication occurs at Mongo DB level.
