    Introduction

    Oncoscape is a web application that hosts an integrated suite of analysis tools for users to explore hypotheses related to molecular and clinical data in order to better understand cancer biology and treatment options. For more detailed information, please read wiki.

    Authentication

    Oncoscape API authentication occurs at Mongo DB level.

    Database Description

    We use two collections to track the metadata for all the data collections within our database. ‘manifest’ is organized by collection. 'lookup_oncoscape_datasource’ is organized by disease.
